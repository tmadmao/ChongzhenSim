# 五、1627开局：加入→退出 最简流程图（可做UI逻辑）
1. 开局：阉党掌权 → 魏忠贤、客氏在任
2. 玩家选择：杀/留魏忠贤
   - 杀 → 阉党退出（失势/灭亡）→ 东林党加入（上台）
   - 留 → 党争暴涨 → 1628秋自动政变
3. 陕西灾荒 → 民变事件加入
   - 赈灾 → 民心上升，流寇延迟加入
   - 不赈灾 → 高迎祥→李自成 依次加入
4. 关外压力 → 袁崇焕登场
   - 重用 → 关宁军势力强化
   - 赐死 → 关宁军忠诚度暴跌，可能退出/兵变

# 六、你开发可直接复制的【核心规则条例】
1. **势力不在 → 人物不登场；人物不在 → 事件不触发**
2. **1627年唯一必亡势力：阉党（玩家不杀，系统也会杀）**
3. **流寇、李自成、后金 不会自动退出，只能被消灭**
4. **人物一旦赐死/叛变，永不复出；势力灭亡，永不重生**
5. **所有事件、人物、势力的加入退出，只看：威望、忠诚、支持度、时间**
